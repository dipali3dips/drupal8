INTRODUCTION
------------

The Menu Title Length allows to change the length
of the menu title field.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/menu_title_length


REQUIREMENTS
------------

This module requires no modules outside of Drupal core.


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit:
   https://www.drupal.org/documentation/install/modules-themes/modules-8 for
   further information.


CONFIGURATION
-------------

If you want to change the size of menu title field , add 
$settings['menu_title_length'] = NUMBER_OF_CHARACTERS; 
in the settings.php before installation. 
Defualt this value is 20 in this module.
